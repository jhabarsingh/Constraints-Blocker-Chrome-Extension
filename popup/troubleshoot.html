Constraints Blocker isn't working?

First, try the following:
- refresh the Leetcode or binarysearch page
- close all tabs and restart the browser
- disable any other extensions
- disable and re-enable the extension
- uninstall and reinstall Extension

If the problem persists, then add issue at here or mail me at sumantopal07@gmail.com

To help me help you faster, include the following:
- description of the issue
- screenshot or video
- browser (ex: Chrome 88.0)
- version of Extension (ex: 1.5.1)
